Calculator Application
A clean, responsive, and functional calculator built using core web technologies. This project serves as a foundational exercise in DOM manipulation, event handling, and CSS layout design.

🚀 Features
Core Arithmetic: Supports addition, subtraction, multiplication, and division.

Intuitive UI: A clean, user-friendly interface designed for ease of use.

Responsive Design: Fully adaptive layout that works seamlessly across desktop, tablet, and mobile devices.

🛠️ Technologies Used
HTML: Structured the calculator layout using semantic elements.

CSS: Styled the interface using modern techniques for responsiveness.

JavaScript: Implemented the calculation logic, input handling, and event listeners.
